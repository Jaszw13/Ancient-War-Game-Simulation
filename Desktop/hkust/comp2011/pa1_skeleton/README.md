## Setup & Compilation (Mac/Linux)
1. Clone this repository:
   ```bash
   git clone https://github.com/YourGitHubUsername/pa1_skeleton.git
   cd pa1_skeleton
   ```
2. Compile the code:
   ```bash
   # Use Makefile (if available)
   make
   # Or manual compilation
   gcc -o pa1 main.c utils.c -Wall
   ```
3. Run the program:
   ```bash
   ./pa1
   ```
